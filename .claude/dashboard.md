## 使い方

このファイルは各 Claude Code インスタンスが自動更新します。

### タスク開始時
1. `backlog/` からタスクを取得
2. `in-progress/[instance-name]/` に移動
3. このダッシュボードの「実装中タスク」セクションを更新

### タスク完了時
1. `in-progress/` から `review/` に移動
2. このダッシュボードの「レビュー待ちタスク」セクションを更新

### レビュー完了時
1. `review/` から `completed/` に移動
2. このダッシュボードの「完了タスク」セクションを更新

### 更新方法
- 各セクションのコメント例を参考に、テーブル形式で記載
- 最終更新日時を必ず更新
- 全体サマリーの数値も更新
